# Change Log
All notable changes to the Topas extension will be documented in this file.

## [1.0.0] - 2019-01-02
### Added
- Syntax highlighting
- Integration with VS Code's commenting, bracketing, and surrounding systems
  
## [2.0.0] - 2026-05-22
### Added
- IntelliSense for TOPAS parameters, including parameter names, values, and documentation
- Hover documentation for parameters
- Code formatting for TOPAS files
- Code folding based on TOPAS block structure
- Outline view for TOPAS files
- Integration with VS Code's task system to run TOPAS simulations directly from the editor
- ISC license file and VSIX packaging exclusions
